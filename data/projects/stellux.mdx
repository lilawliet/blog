## 项目概述

Stellux 交易系统多端项目，涵盖了客户端、管理端、H5 以及 APP 的开发。项目采用 ant design、umijs、tradingview、react-native、react 等技术栈，致力于打造高性能、易用且多端适配的外汇交易系统。

### 技术及体系

    - Ant Design
    - UmiJS
    - TradingView
    - React Native
    - React
    - TypeScript
    - 多端适配

### 核心功能

- 多端统一账户体系
- 实时行情展示与交易
- 订单管理与历史查询
- 资金管理与出入金
- 多种图表与技术指标支持
- 管理后台权限与风控
- 客户端、H5、APP 全端覆盖
- 高性能数据推送与刷新
- 多语言与主题切换
